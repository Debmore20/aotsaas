### UNX-TITAN

## Project

The vision of UNX-TITAN, a "SAAS" webapp, is to convert real life portraits and selfies into Titans from the A.O.T series! Join me on the journey of bringing this idea to reality :)

## Stack

React / Next JS
TypeScript
Tailwind Css
Prisma / MySql ddatabse

## API

NextJS - App router
RESTful
Zod (for validation)
React-query

## UI

Radix UI
